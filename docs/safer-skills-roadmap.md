# Safer Skills Roadmap

The next expansion path should prioritize broader, lower-risk skills before more advanced procedures.

## Why This Direction

- broader value across nursing, allied health, community health, and low-resource training
- safer simulation scope than advanced invasive procedures
- easier faculty review and rubric authoring
- more practical for public-good and access-oriented deployments

## Recommended Module Order

1. Sterile technique
2. Wound dressing
3. PPE donning and doffing
4. Hand hygiene
5. Basic instrument handling

## Design Principles

- simulation-only by default
- visible, faculty-reviewable cues
- simple practice surfaces and inexpensive materials
- plain-language feedback that works well in multilingual and lower-bandwidth settings
- explicit safety refusal paths when images look clinical or patient-facing
